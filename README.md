# Bagas Rudianto — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 Deploy ke GitHub Pages

1. Upload semua file ini ke repository GitHub kamu
2. Buka **Settings** → **Pages**
3. Di bagian **Source**, pilih branch `main` → folder `/ (root)`
4. Klik **Save**
5. Tunggu 1–2 menit, lalu buka: `https://<username>.github.io/<repo-name>/`

## 📁 Struktur File

```
├── index.html        ← halaman utama
├── css/
│   └── style.css     ← semua styling
├── js/
│   └── main.js       ← semua animasi & interaksi
├── assets/
│   └── photo.jpg     ← foto profil
└── .nojekyll         ← agar GitHub Pages tidak error
```
